# SMTS-artifact

## Usage example
```
python getUnsolved.py
-l <Filter out benchmarks based on logic logics (semicolon
                        separated) (default: all logics)> : QF_LRA,... 
-f <Filter out benchmarks based on csv file> : Path to csv file
-r <Filter out benchmarks based on result> : sat | unsat | starexec-unknown
-s <Filter out benchmarks based on solver> : cubeandconquer_default | portfolio_default
```
